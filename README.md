# FlapBirds

Projeto simples inspirado em Flappy Bird, feito em Python com Pygame.

## Requisitos

- Python 3.10+ (recomendado)
- `pygame`

## Instalação

```bash
pip install pygame
```

## Como executar

No terminal, dentro da pasta do projeto:

```bash
python FlapBird.py
```

## Controles

- `Espaço`: faz o pássaro pular e inicia o jogo
- Fechar janela: encerra o jogo

## Estrutura

- `FlapBird.py`: código principal do jogo
- `imgs/`: sprites e imagens usadas no jogo

## Objetivo

Passe pelos canos sem colidir para aumentar a pontuação.
